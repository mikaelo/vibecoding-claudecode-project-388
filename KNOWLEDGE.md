# KNOWLEDGE.md

Единый источник настроек пайплайна отслеживания цен портфеля Финам. **Все
skill'ы (`extract_price`, `tracker`, `alert`) берут значения по умолчанию
отсюда.** Явно переданные skill'у аргументы имеют приоритет над значениями из
этого файла. Чтобы изменить поведение пайплайна — правь этот файл, а не skill'ы.

## Правило алерта

- `threshold` (порог срабатывания) — **`0.10`** (10%).
- `direction` (направление) — **`both`** (алерт и при росте, и при падении).
- Условие срабатывания по бумаге:
  `abs(current − averagePrice) / averagePrice ≥ threshold`,
  где `current` — текущая цена (из `extract_price`), `averagePrice` — средняя
  цена покупки (из Finam MCP `get-account`).
- Направление элемента: `down` при `current < averagePrice`, `up` при
  `current > averagePrice`. При `direction=down` учитываются только падения, при
  `direction=up` — только рост, при `both` — любое отклонение ≥ порога.

## Счета по умолчанию

- `account` — **`869168`**. Если счёт не передан явно, используется этот.

## Хранилище snapshot (репозиторий данных)

- `repo` — **`mikaelo/tracker-data`**.
- `branch` — **`main`**. Snapshot'ы пишутся **напрямую в `main`**, без создания
  веток и без PR.
- Имя файла — **`YYYY-MM-DD-HH-MM.json`** в корне репозитория; дата/время берутся
  из `generated_at` в **UTC** (напр. `2026-07-14-14-15.json`).
- Запись — GitHub MCP `create_or_update_file`. Имя уникально в пределах минуты →
  это всегда новый файл, `sha` не нужен.

## Котировки и формат

- Источник котировок — Finam MCP `get-quote`.
- Биржа по умолчанию — **`@MISX`** (Московская биржа). Если в тикере нет `@MIC`,
  добавляется `@MISX`: `SBER` → `SBER@MISX`.
- Время в колонках/полях `datetime` — **московское (МСК, UTC+3)** в формате
  ISO 8601 со смещением `+03:00` (время котировки `as_of` из ответа `get-quote`
  приходит в UTC и переводится в МСК, +3 часа).
- `generated_at` в snapshot — в **UTC** (из него строится имя файла).

## Telegram

- `bot_token` — из переменной окружения **`TELEGRAM_BOT_TOKEN`**.
- `chat_id` — из переменной окружения **`TELEGRAM_CHAT_ID`**.
- Отправка — Telegram Bot API `sendMessage`. Если хотя бы один доступ не задан,
  сообщение не отправляется, а результат помечается `telegram.sent=false`.
- Секреты (сам токен) в этот файл и в вывод skill'ов не попадают — здесь только
  **имена** переменных окружения.
